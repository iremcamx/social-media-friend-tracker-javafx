# Sosyal Medya Arkadaş Takibi (JavaFX)

Java OOP dersi için hazırlanmış, JavaFX arayüzü üzerinden kullanıcı ekleme, arkadaş ekleme/çıkarma ve takip etme/takipten çıkma işlemlerini yapan örnek uygulama.

## Özellikler
- Kullanıcı ekleme (User / Influencer)
- Arkadaş ekleme/çıkarma (çift yönlü)
- Takip etme/takipten çıkma (tek yönlü)
- Influencer takipçi sayacı (polymorphism + instanceof)

## Gereksinimler
- Java 17+
- Maven
- JavaFX (pom.xml üzerinden gelir)

## Çalıştırma
IntelliJ:
1) `pom.xml` açılır, Maven import edilir
2) Terminal: `mvn clean javafx:run`
veya
3) Main sınıfı: `com.example.friendtracker.MainApp`
